Features:

  • secure: no login or password required. Use your API key and subdomain.
  • select rooms to sync
  • swipe left and right to change rooms
  • queued uploads when your device is offline
  • change sync settings based on network: wifi or mobile

Permissions:

  • full network access. There is also an anonymous crash report uploader that uploads the nature of the crash without sending any personally identifying data. Also the Google Analytics mobile library is used to track anonymous and aggregated use statistics.
  • network state: the app needs to determine the network state and which network type is on (wifi or mobile).
This is a beta release and your feedback is greatly appreciated. There is a public trello board and there are two feedback mechanism in the app.
Feedback is a gift. Please be generous.
